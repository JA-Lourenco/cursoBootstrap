Programação

O que é?

A programação pode ser resumida em uma pequena frase, porém de grande efeito.

Permite resolver problemas através da tecnologia, ou seja, por exemplo, possibilitar maior produtividade/acessibilidade em uma atividade rotineira.

O que preciso saber?

Lógica de Programação

Neste tópico, é de extrema importância que você estude e compreenda da melhor forma possível. A lógica de programação te permitirá a organizar tudo de forma coesa dentro do código.

Estrutura de Dados

É uma estrutura organizada de dados na memória de um computador ou em qualquer dispositivo de armazenamento, de forma que os dados possam ser utilizados de forma correta. Exemplos: Vetor, Matriz, Registro, etc.

Linguagem de Programação

É uma linguagem escrita e formal que determina um conjunto de instruções. Escritas pelo programador para o computador executar alguma ação. Para se utilizar, é importante ter os outros dois conceitos bem definidos, permitindo melhor aplicação deste.

Caminhos - Áreas de atuação

Front-end

É a programação Client-side, ou seja, o Desenvolvedor atua na aplicação do lado do cliente. Isso significa que, você trabalhará desenvolvendo as funcionalidades que terão contato direto com os usuários. Assim como a esse texto que está realizando leitura neste momento, as imagens vistas no site e até aquela barra de navegação no topo da página que pode te levar aos tópicos rapidamente.

Back-end

Nesta posição, o(a) programador(a) terá atuação do lado do servidor, também chamado de Server-side. Este profissional será responsável por organizar e enviar todos os dados para o Front-end, assim permitindo-o a mostrar aos usuários. Geralmente, essa comunicação se deve ao que chamamos de APIs (Application Programming Interface).

Full Stack

E no caso do Full Stack, é simplesmente o profissional que atua nas duas primeiras categorias que vimos. A pessoa possui domínio desde a constituição da informação através do banco de dados, até o momento de desenvolver uma interface para que essa mesma informação seja disponibilizada aos usuários.
